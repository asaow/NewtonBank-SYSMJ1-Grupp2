Det finns 2 alternative för att köra applikationen
1) kör som en executable jar
2) pull ner projekt och och köra i eclipse


Alternativ 1
 1. ladda ner filer/NewtonBank-SYSMJ1-Grupp2.jar från repo
 2. ladda ner filer/CreateDBScript.sql och kör scripten i MySQLWorkbench för att skapa NewtonBank databasen
 3. Kör applicationen(jar filen) och testa de olika funktionalitet
 
Alternativ 2
 1. ladda ner filer/CreateDBScript.sql och kör scripten i MySQLWorkbench för att skapa NewtonBank databasen
 2. clona ner projektet från denna repo i din hårddisk
 3. skapa ett Java projekt i eclipse och välj den clonade mappen som destination directory
 4. lägg till JUnit, MySQL JBDC i build path
 5. köra MainFrame.java som är applicationen GUI
 6. Testa funktionalitet i GUI
